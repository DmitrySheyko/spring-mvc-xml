Формат выражения для поиска и удаления книг: [author]/[book title]/[size]

*/*/* - удалить все книги.

Иванов/*/* - удалить все книги автора Иванов.

*/Книга для чайников/* - удалить все книги с названием "Книга для чайников".

*/*/=100 - удалить все книги с количеством страниц 100".

*/*/>100 - удалить все книги с количеством страниц более 100".
